Файлы 'dashbrd_action.csv', 'Задание_аналитик.xlsx', 'Задание_аналитик.txt' - исходные материалы задачи.
Файл 'Тестовое задание Волков Т.Ф..ipynb' - "тетрадка" с последовательным решением задач. Одноименный файл .html предназначен для быстрого ознакомления с содержимым тетрадки.
Файлы task_X.py - скрипты, выполняющие поставленные задачи при их запуске. Код перенесён из тетрадки, отформатирован в подобающий вид с исключением промежуточной аналитики.
Файлы task_X_result.xlsx - результаты выполнения соответствующих скриптов task_X.py
